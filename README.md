# Task-Manager
## Features
- Add / delete tasks
- Mark as completed / active
- Search & filter tasks
- Task details screen
- Local storage (AsyncStorage)
- Fetch initial data from public API

## API
Using DummyJSON:
https://dummyjson.com/todos

## Tech Stack
- React Native (Hooks)
- JavaScript
- AsyncStorage
- Fetch API
- React Navigation

## Setup
```bash
git clone https://github.com/altiinbb/Task-Manager.git
cd task-app
npm install
npm start
